# Mobile App Project
## Team HogRida (ID: 38) presents
## Scavenger Hunt!

## Game Overview
Welcome to HogRida's Scavenger Hunt! The goal of the game is to find objects that are prompted by the game and take a picture of it! If the current object is way out of reach, no worries! 
Choosing an picture from your gallery also works but a Hunt should be adventurous so go out and find the objects! This is game is a simple and fun way to start moving about and adding 
the "thrill of the Hunt" integrated in order everyday lives. Play alone and beat your previous highscores or challenge your friends to see who is the better Scavenger Hunter! All the best and have fun!

### Logging In
To log in ensure you have an account registered, if there is no account create one by typing in a username and password and click Register.
If an account has already been created enter the credentials and press LogIn

### Getting Started
Once Logged in there will be a Home Page that shows the "Start Game" Button, the "HighScore" Button, the "How to Play" button and lastly the Log Out button.

### Start Game
When the "Start Game" button is pressed it will bring you to a new screen that shows either "Mystery Hunt"  or "Category Shuffle". Press either one to start the game.

#### Mystery Hunt
When the "Mystery Hunt" button is pressed it will bring you to a category selection screen to choose various categories of items and objects to find. After selecting a category 
the Game will start and a countdown timer will begin. This Game Mode will not explicitly give the items or objects to find but will instead give hints and the player must derive
which object to find within that category in order to gain the points. Score points by either choosing a photo from the gallery or taking a picture in real time. Skipping the current object will consume points. 
The Goal is to get as many points as possible within the given time frame.

#### Category Shuffle
When the "Category Shuffle" button is pressed it will bring you to a category selection screen to choose various categories of items and objects to find. After selecting a category 
the Game will start and a countdown timer will begin. This Game Mode will  give the items or objects to find and the player must search for that object in order to gain points. After every
object found the next category will be randomized. Score points by either choosing a photo from the gallery or taking a picture in real time. Skipping the current object will consume points. 
The Goal is to get as many points as possible within the given time frame.

### HighScore
When the "HighScore" button is pressed it will bring you to a HighScore page where the player can view their past socres.
The Highscores can be sorted and displayed based on previous completed games.

### How To Play
When the "How to Play" button is pressed it will bring you to a How to Play page where the player can view how the game works and the mechanics of the game. The various game modes,
how to navigate and the controls of the Game.
